# VitDaily

VitDaily는 업비트를 통해 비트코인 정기 매수를 예약하고 자동으로 실행할 수 있는 데스크톱 앱입니다. 프론트엔드는 Tauri, React, TypeScript, Tailwind CSS로 구성되어 있으며, 실제 매수 스케줄러와 업비트 API 연동은 Rust 백엔드에서 처리합니다.

## 주요 기능

- **업비트 API 키 관리**  
  업비트 Access Key와 Secret Key를 앱에 저장하고 관리할 수 있습니다. 저장된 API 키는 평문 파일이 아니라 운영체제의 키체인에 보관됩니다.

- **API 연결 테스트**  
  자동 매수를 사용하기 전에 입력한 업비트 API 키가 정상적으로 동작하는지 확인할 수 있습니다. 연결 테스트를 통해 키 저장 여부와 업비트 계정 API 접근 가능 여부를 확인합니다.

- **일별 KRW-BTC 매수 스케줄 관리**  
  매일 지정한 시간에 지정한 금액만큼 KRW-BTC를 매수하도록 스케줄을 등록, 수정, 삭제, 활성화 또는 비활성화할 수 있습니다.

- **최소 주문 금액 검증**  
  업비트의 최소 주문 금액인 5,000원 미만의 매수 스케줄은 저장되지 않도록 검증합니다.

- **수정 사항 적용 시점 선택**  
  기존 스케줄의 시간이나 금액을 수정할 때 변경 사항을 내일부터 적용할지, 오늘 바로 반영할지 선택할 수 있습니다. 기본적으로는 실수 방지를 위해 내일부터 반영하는 흐름을 사용합니다.

- **백그라운드 스케줄러**  
  Rust 백엔드에서 30초마다 실행할 스케줄이 있는지 확인합니다. 같은 스케줄이 하루에 중복 실행되지 않도록 매수 로그를 기준으로 실행 여부를 검사합니다.

- **업비트 시장가 매수 실행**  
  실행 시간이 된 스케줄은 업비트 주문 API를 통해 KRW-BTC 시장가 매수 주문으로 처리됩니다.

- **로컬 매수 내역 기록**  
  매수 성공 또는 실패 결과를 로컬 로그로 저장합니다. 실패한 경우에는 오류 메시지도 함께 기록해 원인을 확인할 수 있습니다.

- **오늘의 매수 요약**  
  대시보드에서 오늘 성공한 매수 금액과 매수된 BTC 수량을 요약해서 확인할 수 있습니다.

- **시스템 알림 설정**  
  매수 성공 또는 실패 시 운영체제의 시스템 알림을 받을 수 있으며, 설정 화면에서 알림 사용 여부를 변경할 수 있습니다.

## 시작하기

의존성을 설치합니다:

```sh
npm install
```

프론트엔드 개발 서버를 실행합니다:

```sh
npm run dev
```

Tauri 데스크톱 앱을 실행합니다:

```sh
npm run tauri dev
```

프론트엔드를 빌드합니다:

```sh
npm run build
```

Rust 백엔드를 확인합니다:

```sh
cd src-tauri
cargo check
```

## 프로젝트 구조

```text
src/
  App.tsx                  React 앱 셸과 화면 전환
  components/              대시보드, 설정, 스케줄, 매수 로그 UI
  types/                   프론트엔드 공통 TypeScript 타입
src-tauri/
  src/commands.rs          Tauri 명령, 스케줄러, 업비트 API 호출, 로컬 저장소 처리
  src/types.rs             백엔드 데이터 모델
  src/lib.rs               Tauri 앱 설정과 명령 등록
  tauri.conf.json          Tauri 설정 파일
```

## 로컬 데이터 저장

VitDaily는 사용자 데이터를 로컬 머신에 저장합니다.

- API 인증 정보는 운영체제 키체인에 저장됩니다.
- 매수 스케줄은 `schedules.json`에 저장됩니다.
- 매수 로그는 `logs.json`에 저장됩니다.
- 앱 설정은 `settings.json`에 저장됩니다.

JSON 파일은 각 운영체제의 로컬 데이터 디렉터리 아래 `vitdaily` 폴더에 저장됩니다.

## 주의사항

VitDaily는 유효한 업비트 API 키와 활성화된 스케줄이 설정되어 있으면 실제 업비트 시장가 매수 주문을 실행합니다. API 키는 필요한 최소 권한만 부여해 사용하고, 앱을 실행 상태로 두기 전에 스케줄 시간과 금액을 반드시 확인하세요.
